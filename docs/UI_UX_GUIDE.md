# UI / UX Guide

## UI language

All user-facing UI text must be Turkish.

## Product feel

Clean, fast, modern and motivating.

Avoid cheap gamification.

## Dashboard must answer

Within 3 seconds, user should understand:

- How many calories today?
- How many calories left?
- Macro status?
- Steps?
- Workout/off-day status?
- Weekly rank?
- Streak?

## Empty state examples

Good:

```txt
Bugün henüz yemek eklemedin. İlk öğününü ekleyerek takibe başla.
```

Bad:

```txt
No data
```
